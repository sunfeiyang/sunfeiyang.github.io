# 设置启动时不自动打开上次项目

  • 打开 File > Setting (快捷键：Ctrl+Alt+S)

  • 打开 Appearance & Behavior - System Setting
    - 去勾选 Reopen last project on startup （启动时重新打开上一个项目）
