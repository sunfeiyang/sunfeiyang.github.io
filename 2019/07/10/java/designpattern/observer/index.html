java设计模式之观察者模式

2019-07-10 Java 466 字
设计模式 Spring

Java 设计模式，眼观六路，耳听八方，观察者很忙，随时监控着关注对象的一举一动。观察者模式解决了基于一对多对象结构关系间的互动问题，使观察者（多方买家）专主动为被动，被观察者（单方商家）转被动为主动，此情此景，需动静结合，先后有序，以【一方动多方静】取代【多方动一方静】的模式，大大的提高了沟通效率。
