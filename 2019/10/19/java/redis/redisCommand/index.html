# 优缺点

优点
1）同类数据归类整合储存，方便数据管理
2）相比 string 操作消耗内存与 CPU 更小
3）相比 string 储存更节省空间

缺点
1）过期功能不能使用在 field 上，只能用在 key 上
2）Redis 集群架构下不适合大规模使用
