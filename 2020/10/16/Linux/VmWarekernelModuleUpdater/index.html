Linux系统下VMware kernel Module Updater弹框，Install失败

Linux VMware 2020/10/16

Ubuntu 系统更新后，打开 VMware 提示需要更新安装，点击安装失败问题解决

  1. 提示框

  2. 安装失败

    --处理方法
    --打开终端执行如下代码
    VMWARE_VERSION=workstation-15.5.1
    TMP_FOLDER=/tmp/patch-vmware
    rm -fdr $TMP_FOLDER
    mkdir -p $TMP_FOLDER
    cd $TMP_FOLDER
    git clone https://github.com/mkubecek/vmware-host-modules.git
    cd $TMP_FOLDER/vmware-host-modules
    git checkout $VMWARE_VERSION
    git fetch
    make
    -- 执行如下代码时需要输入管理员密码，输入完成后执行当前代码，但是后续的代码会终止，需要手动拷贝未执行其余命令继续操作，全部命令执行完成后，VMware即可正常使用
    sudo make install
    sudo rm /usr/lib/vmware/lib/libz.so.1/libz.so.1
    sudo ln -s /lib/x86_64-linux-gnu/libz.so.1
    /usr/lib/vmware/lib/libz.so.1/libz.so.1
    sudo /etc/init.d/vmware restart
