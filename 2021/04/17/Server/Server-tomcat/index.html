tomcat 配置 gc 日志输出

omcat 目录 /bin

找到 catalina.sh (这是 linux 下，window 下应该是 catalina.bat)

配置 JAVA_OPTS 参数，我这里配置的是 tomcat 目录 /log 目录下

JAVA_OPTS= -Xms1024m -Xmx3048m -XX:PermSize=512m -XX:MaxPermSize=1524m -Xss4096K
-Xloggc:…/logs/tomcat_gc.log
