云服务器80端口被占用

2021-01-27 Server 224 字
Tomcat

在使用案例云或其他厂商的云服务器时遇到的 80 端口被占用问题，如何释放 80 端口？问题主要出现在 Windows Server 服务器上，Linux 服务器上暂且还没有遇到类似情况。

问题：80 端口被占用？如何释放 80 端口
第一步、win+r 输入 cmd
第二步、在控制台输入 netstat -aon | findstr "80" 命令。找出类型为 TCP 端口 80 的，然后记录 PID (最后一行为 PID) 如下图

第三步、在控制台输入 tasklist | findstr “PID 数值”
第四步、打开任务管理器，查看 PID=4 是一个 system 进程，其对应的进程描述是 NT kernel & system。
第五步、win+r 输入 regedit 回车，找到 HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\HTTP ，并修改
Start 的数值数据为 4.（如下图）

第六步、修改注册表后要重启电脑才会生效
